# barebitcoin-go

> ⚠️ WARNING ⚠️: The implementation in this package has not been sufficiently tested, which in the worst case may lead to LOSS OF FUNDS. Proceed at your own risk and preferably use read-only API keys.

An API client library for the Bare Bitcoin API. A CLI is also available.

## Authentication

You can create API keys [here](https://barebitcoin.no/innlogget/profil/nokler).

To use them with the CLI, you need to set these environment variables.

```sh
export BAREBITCOIN_PUBLIC_KEY="<public>"
export BAREBITCOIN_SECRET_KEY="<secret>"
```
